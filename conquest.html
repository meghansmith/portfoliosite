Conquest Brewing

A Student Design & Develop Project

Scope: Create a sleek, modern site that portrays the personality of this local brewery. Edit images and text. This website is all about big photos and typography. I used keyframe animations to fade in the header text and reused that code in the navigation when it appears. To showcase their personality, I focused on making very powerful images and pairing them with bold text. Since modern sites are long and scrolling, I utilized jQuery to initiate a sticky navigation when the site is scrolled.

They needed a way to showcase their various brews. I developed a template for each beer that would work with short or long descriptions. By using a large fixed image with a nearly opaque overlay, I was able to keep the feel of the homepage yet cleanly display a lot of content. I kept the tagline in the header, but displayed it horizontally so the users more quickly get to the content.
